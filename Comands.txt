Склонировать репозиторий с сервера
git clone <ssh>

Запомнить текущее состояние файла (. - всех файлов) и добавить его в текущую ревизию
git add .

Зафиксировать текущую ревизию (локально)
git commit -m "Коммент"

Отправить данные о коммитах(ревизиях) на сервер
git push

Загрузить данные с сервера
git pull

Посмотреть че почем в репозитории
git status
